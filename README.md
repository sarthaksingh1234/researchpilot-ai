# ResearchPilot AI

AI-powered research workspace to upload PDFs, generate summaries, chat with documents, manage projects, search by tags, and export research notes.

## Tech Stack

- React + Vite
- Tailwind CSS
- Node.js + Express
- MongoDB Atlas
- JWT Authentication
- Gemini AI
- PDF Upload & Summarization

## Features

- User authentication
- Project dashboard
- AI research generation
- PDF upload and summarization
- Chat with PDF
- Persistent PDF chat history
- Search and tags
- Markdown export
- PDF export
- Rate limiting and validation
- Responsive UI

## Client 

VITE_API_URL=http://localhost:5000/api

## Deployment 

Backend deployed on Render
Frontend deployed on Vercel
Database hosted on MongoDB Atlas
